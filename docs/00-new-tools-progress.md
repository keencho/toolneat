## 기술 스택
- **PDF**: pdf-lib, PDF.js
- **AI 배경제거**: @mediapipe/selfie_segmentation
- **비디오→GIF**: FFmpeg.wasm
- **OCR**: Tesseract.js
- **화면 녹화**: MediaRecorder API (네이티브)

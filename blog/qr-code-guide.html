무료 QR코드 만들기: 완벽 가이드 2025

2025년 1월 25일 · 7분 읽기

식당 메뉴판, 명함, 제품 포장지... 요즘 QR코드가 없는 곳이 없죠. "나도 만들어볼까?" 싶은데 어디서 어떻게 만드는지 막막하셨다면, 이 글이 도움이 될 거예요.

QR코드가 뭔가요?

QR코드는 Quick Response Code의 약자입니다. 1994년 일본 덴소 웨이브에서 자동차 부품 추적용으로 개발했는데, 지금은 전 세계적으로 사용되고 있죠.

기존 바코드가 약 20자 정도만 저장할 수 있는 반면, QR코드는 숫자 7,089자, 문자 4,296자까지 담을 수 있습니다. 그래서 URL, 연락처, 와이파이 정보 같은 복잡한 데이터도 저장 가능해요.

QR코드로 뭘 할 수 있나요?

가장 흔한 활용 사례들을 정리해봤습니다:

  • 웹사이트 연결: 링크를 직접 입력하지 않아도 스캔만으로 이동
  • 와이파이 공유: 비밀번호 불러줄 필요 없이 스캔으로 연결
  • 연락처 저장: 명함 대신 스캔으로 전화번호부에 바로 추가
  • 결제: 카카오페이, 네이버페이 등 간편결제
  • 출입 인증: 건물 출입, 이벤트 티켓 확인
  • 메뉴판: 식당에서 종이 메뉴 대신 디지털 메뉴 제공

무료로 QR코드 만드는 방법

QR코드 생성은 생각보다 간단합니다. 몇 가지 방법을 소개할게요.

1. 온라인 무료 도구 사용

가장 쉽고 빠른 방법입니다. 별도 프로그램 설치 없이 브라우저에서 바로 생성할 수 있어요.

사용 방법:

  1. QR코드 생성 도구에 접속
  2. 담고 싶은 내용 (URL, 텍스트 등) 입력
  3. "생성" 버튼 클릭
  4. PNG 또는 SVG로 다운로드

2. 크롬 브라우저 내장 기능

크롬 브라우저에는 현재 페이지의 QR코드를 생성하는 기능이 내장되어 있습니다.

주소창 오른쪽의 공유 아이콘을 클릭하고 "QR 코드 만들기"를 선택하면 됩니다. 단, 현재 보고 있는 페이지 URL만 가능하다는 제한이 있어요.

QR코드 유형별 만들기

URL QR코드

가장 기본적인 유형입니다. 웹사이트, 유튜브 영상, SNS 프로필 등 모든 링크를 QR코드로 만들 수 있어요.

팁: URL 단축 서비스(bit.ly 등)로 먼저 주소를 줄이면 QR코드가 더 단순해지고, 스캔 인식률이 높아집니다.

와이파이 QR코드

손님이 오면 와이파이 비밀번호 알려주기 귀찮죠? 와이파이 QR코드를 만들어두면 스캔만으로 자동 연결됩니다.

WIFI:T:WPA;S:네트워크이름;P:비밀번호;;

이 형식으로 입력하면 되는데, 대부분의 QR코드 생성기에서 와이파이 옵션을 선택하면 자동으로 만들어줍니다.

연락처(vCard) QR코드

명함에 QR코드를 넣으면 상대방이 일일이 연락처를 입력할 필요가 없어요. 이름, 전화번호, 이메일, 회사 정보까지 한 번에 저장됩니다.

좋은 QR코드 만드는 팁

1. 적절한 크기

QR코드가 너무 작으면 스캔이 안 됩니다. 최소 2cm x 2cm 이상을 권장해요. 멀리서 스캔해야 한다면 (포스터, 광고판 등) 그만큼 크기를 키워야 합니다.

2. 적절한 여백

QR코드 주변에는 "조용한 영역(Quiet Zone)"이라 불리는 여백이 필요합니다. 최소 4칸(모듈) 이상의 흰색 여백을 남겨두세요.

3. 대비 유지

어두운 패턴 + 밝은 배경이 기본입니다. 색상을 바꿀 수 있지만, 대비가 충분하지 않으면 스캔이 잘 안 돼요. 특히 파스텔톤 색상은 피하세요.

4. 테스트 필수

인쇄하기 전에 반드시 여러 기기에서 스캔 테스트를 해보세요. 특히:

  • 아이폰 기본 카메라
  • 안드로이드 기본 카메라
  • 조명이 어두운 환경

PNG vs SVG, 뭘로 저장할까?

형식 장점 추천 용도
PNG 호환성 좋음, 어디서나 사용 가능 웹, SNS, 일반 인쇄
SVG 확대해도 깨지지 않음 대형 인쇄물, 고해상도 필요시

일반적인 용도라면 PNG로 충분합니다. 배너나 간판처럼 크게 인쇄해야 한다면 SVG를 선택하세요.

자주 묻는 질문

Q: QR코드에 유효기간이 있나요?

정적 QR코드(일반적인 것)는 영구적입니다. 단, 동적 QR코드(단축 URL 서비스 사용)는 서비스가 종료되면 작동하지 않을 수 있어요.

Q: QR코드 안에 로고를 넣어도 되나요?

QR코드는 최대 30%까지 손상되어도 복원 가능하도록 설계되어 있어요. 그래서 중앙에 작은 로고를 넣어도 대부분 스캔됩니다. 단, 반드시 테스트 후 사용하세요.

Q: 색상을 바꿔도 스캔이 되나요?

네, 단 어두운 색 + 밝은 배경 조합을 유지해야 합니다. 반전(밝은 패턴 + 어두운 배경)은 일부 스캐너에서 인식이 안 될 수 있어요.

마무리

QR코드 만들기, 생각보다 어렵지 않죠? 핵심만 정리하면:

  • 온라인 무료 도구로 쉽게 생성 가능
  • 충분한 크기와 여백 확보
  • 대비 유지 (어두운 패턴 + 밝은 배경)
  • 인쇄 전 테스트 필수

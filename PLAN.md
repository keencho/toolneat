# Toolneat - 프로젝트 계획

## 개요
깔끔하고 가벼운 온라인 도구 모음 사이트. 한국어/영어 지원 (추후 중국어/일본어 추가 예정).

---

## 1. 기술 스택

### 확정 사항
- **HTML/CSS/JS**: 생짜 (프레임워크 없음)
- **CSS**: Tailwind CLI v4 (사용한 클래스만 빌드)
- **폰트**: Pretendard (로컬 호스팅)
- **아이콘**: SVG (인라인, 다크모드 대응 가능)
- **이미지**: WebP (OG 이미지, 썸네일용)
- **서버**: 없음 (정적 사이트)
- **다국어**: 단일 파일 + `?lang=en` 쿼리 (파일 중복 없음)
- **도메인**: toolneat.com

### 빌드 명령어
```bash
npm run dev    # Tailwind watch + serve (동시 실행)
npm run build  # 배포용 CSS 빌드 (minified)
```

    └── Cron 표현식

├── 계산기
│   ├── 연봉 실수령액
│   ├── 대출 이자
│   └── BMI

**PC 레이아웃**:
- 메가 메뉴 (hover시 드롭다운)

- 비슷한 도구끼리 그룹화

**모바일 레이아웃**:
- 아코디언 스타일

## 5. 다국어 확장 계획

### 현재 지원
- 한국어 (기본)
- 영어 (`?lang=en`)

### 추후 추가 예정
- 일본어 (`?lang=ja`)
- 중국어 간체 (`?lang=zh-CN`)
- 중국어 번체 (`?lang=zh-TW`)

### i18n 개발 가이드라인
1. 모든 동적 텍스트는 `t('key')` 사용
2. 언어 변경 시 `i18nApplied` 이벤트 리스너로 업데이트
3. 날짜/숫자 포맷은 언어별 처리
4. `window.switchLanguage` 오버라이드 금지

## 6. 개발 우선순위 제안

### 즉시 진행 (수요 높음 + 난이도 낮음)
1. **색상 변환기** - 개발자 필수, 구현 간단
2. **Hash 생성기** - Web Crypto API 활용, 간단
3. **BMI 계산기** - 공식 단순, 범용성 높음

### 단기 (1-2주)
4. **QR 코드 생성기** - 라이브러리 활용 (qrcode.js)
5. **정규표현식 테스터** - 수요 높음, 중간 난이도
6. **대출 이자 계산기** - 한국 특화, 실용적

### 중기 (2-4주)
7. **Cron 표현식 생성기** - 개발자 도구로 인기
8. **Markdown 에디터** - 콘텐츠 제작자 필수
9. **이미지 리사이저** - Canvas API 활용

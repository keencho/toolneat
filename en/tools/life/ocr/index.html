What is OCR?

OCR (Optical Character Recognition) recognizes text in images. Extract text from scanned documents, screenshots, and photos.

How to Use

  1. Select the language to recognize (English, Korean, Japanese, etc.)
  2. Drag or click to upload an image
  3. Text is automatically recognized
  4. Click 'Copy' button to copy the text

Frequently Asked Questions

Q: How can I improve accuracy?

Use clear, high-contrast images. Text should be level and have good contrast with the background.

Q: Can it recognize handwriting?

Neat handwriting can be recognized to some extent, but accuracy is lower than printed text.

Q: Why does it take time to process?

Language data (10-30MB) downloads on first use. After that, it's cached for faster processing.

Q: What about images with multiple languages?

Select multiple languages like 'English+Korean' to recognize mixed-language text.

All processing is done in your browser. No data uploaded to servers.

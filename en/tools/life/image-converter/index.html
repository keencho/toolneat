Image Format Guide

  • PNG: Supports transparency, lossless. Best for logos, screenshots
  • JPG: Optimized for photos, small file size. No transparency
  • WEBP: Modern format, 30% smaller than PNG/JPG. Recommended for web
  • GIF: 256 colors, animation support. Best for simple graphics
  • BMP: Uncompressed, large file size. For editing
  • ICO: Windows icon format. Used for favicons

인식된 텍스트

OCR이란?

OCR(Optical Character Recognition)은 이미지에서 텍스트를 인식하는 기술입니다. 스캔 문서, 스크린샷, 사진 등에서 텍스트를 추출할 수 있습니다.

사용 방법

  1. 인식할 언어를 선택합니다 (한국어, 영어, 일본어 등)
  2. 이미지를 드래그하거나 클릭하여 업로드합니다
  3. 자동으로 텍스트가 인식됩니다
  4. '복사' 버튼으로 텍스트를 복사합니다

자주 묻는 질문

Q: 인식률을 높이려면?

선명하고 대비가 높은 이미지가 좋습니다. 텍스트가 기울어지지 않고, 배경과 글자 색상 차이가 클수록 인식률이 높습니다.

Q: 손글씨도 인식되나요?

깔끔한 손글씨는 어느 정도 인식되지만, 인쇄된 텍스트보다 정확도가 낮습니다.

Q: 처리 시간이 왜 오래 걸리나요?

최초 실행 시 언어 데이터(10-30MB)를 다운로드합니다. 이후에는 캐시되어 빠르게 처리됩니다.

Q: 여러 언어가 섞인 이미지는?

'한국어+영어'처럼 여러 언어를 동시에 선택하면 혼합된 텍스트도 인식할 수 있습니다.

모든 처리는 브라우저에서 이루어지며 서버로 전송되지 않습니다.

이미지를 PDF로 변환

여러 이미지를 하나의 PDF 문서로 만듭니다. 스캔 문서 정리, 사진 앨범 제작, 포트폴리오 작성에 유용합니다.

사용 방법

  1. 이미지 파일을 드래그하거나 클릭하여 업로드합니다
  2. 이미지를 드래그하여 원하는 순서로 정렬합니다
  3. 페이지 크기 옵션을 선택합니다
  4. 'PDF 생성' 버튼을 클릭하여 다운로드합니다

자주 묻는 질문

Q: 어떤 이미지 형식을 지원하나요?

PNG, JPG, JPEG, WEBP, GIF, BMP 등 대부분의 이미지 형식을 지원합니다.

Q: 이미지 순서를 변경할 수 있나요?

네, 업로드 후 이미지를 드래그하여 원하는 순서로 재배열할 수 있습니다.

Q: 페이지 크기 옵션은 무엇인가요?

'이미지 크기에 맞춤'은 각 이미지 크기대로 페이지가 생성됩니다. A4, Letter는 해당 용지 크기에 이미지가 맞춰집니다.

Q: 변환 후 이미지 품질이 떨어지나요?

원본 이미지의 품질이 그대로 유지됩니다. 추가 압축 없이 PDF에 포함됩니다.

모든 처리는 브라우저에서 이루어지며 파일이 서버로 전송되지 않습니다.

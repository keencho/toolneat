PDF 병합이란?

PDF 병합은 여러 개의 PDF 파일을 하나의 문서로 합치는 작업입니다. 업무 환경에서 계약서, 보고서, 영수증, 스캔 문서 등을 하나의 파일로 정리해야 하는 경우가 많은데, 이 도구를 사용하면 별도의 프로그램 설치 없이 브라우저에서 빠르게 병합할 수 있습니다. 모든 처리는 사용자의 브라우저 내에서 이루어지므로 파일이 외부 서버로 전송되는 일이 없어 개인정보와 기밀 문서를 안전하게 다룰 수 있습니다.

작동 원리

PDF 병합은 여러 PDF 파일의 페이지 트리를 하나로 결합하는 과정입니다. 각 PDF의 크로스 레퍼런스 테이블을 파싱하여 페이지 객체를 추출하고, 새로운 PDF 문서의 페이지 트리에 순서대로 삽입합니다. 북마크, 주석, 폼 필드 등의 메타데이터도 함께 병합됩니다. 브라우저의 PDF.js 라이브러리를 사용하여 모든 처리가 클라이언트에서 이루어집니다.

병합 순서 따라 하기

  1. 화면 상단의 점선 영역에 PDF 파일을 드래그해서 놓거나, 영역을 클릭한 뒤 파일 선택 창에서 여러 개의 PDF를 한 번에 선택하세요. 라이브러리 로딩이 끝나면 바로 업로드할 수 있습니다.
  2. 빠뜨린 파일이 있다면 선택된 파일 목록 오른쪽 위의 "+ 파일 추가" 버튼을 눌러 언제든지 더 추가할 수 있습니다.
  3. 목록에 표시된 파일을 마우스로 끌어서 원하는 순서로 배치하세요. 목록의 위쪽에 있는 파일일수록 병합된 문서의 앞부분에 들어갑니다.
  4. 순서를 확인했으면 "PDF 병합하기" 버튼을 클릭하세요. 처리가 끝나면 병합된 문서의 총 페이지 수와 파일 크기가 화면에 표시됩니다.
  5. "다운로드" 버튼을 눌러 완성된 PDF를 저장하세요. 다른 문서를 이어서 작업하려면 "새로 병합하기"를 누르면 처음 화면으로 돌아갑니다.

이런 상황에서 활용해 보세요

복합기나 스캐너로 계약서를 스캔하면 페이지별 또는 묶음별로 여러 개의 PDF가 만들어지는 경우가 많습니다. 상대방에게 이메일로 보내기 전에 이 도구로 하나의 파일로 합쳐두면, 받는 사람이 첨부파일을 하나씩 열어볼 필요 없이 전체 계약 내용을 순서대로 확인할 수 있습니다.

취업이나 입찰 서류를 준비할 때도 유용합니다. 이력서, 경력기술서, 자격증 사본, 포트폴리오가 각각 다른 프로그램에서 내보낸 별도의 PDF라면, 제출 전에 요구된 순서대로 배치해 한 파일로 묶어 두는 것이 좋습니다. 심사하는 쪽에서 파일 하나만 열면 되므로 서류가 누락되었다는 오해를 줄일 수 있습니다.

경비 정산도 자주 등장하는 사례입니다. 한 달 동안 모인 카드 영수증, 세금계산서, 거래명세서 PDF를 날짜순으로 정렬해 하나로 병합하면 경리 담당자에게 깔끔하게 전달할 수 있고, 나중에 다시 찾아볼 때도 파일 하나만 보관하면 됩니다.

자주 묻는 질문

파일 개수나 용량에 제한이 있나요?

도구 자체에는 파일 개수 제한이 없습니다. 다만 모든 처리가 브라우저 메모리 안에서 이루어지기 때문에, 수백 MB에 달하는 대용량 파일을 한꺼번에 올리면 기기 사양에 따라 속도가 느려지거나 브라우저가 버벅일 수 있습니다. 아주 큰 문서는 몇 개씩 나눠서 병합하는 것을 권장합니다.

암호가 걸린 PDF도 병합할 수 있나요?

열람 암호가 설정된 PDF는 내용을 읽을 수 없어 병합 과정에서 오류가 발생합니다. 먼저 PDF 뷰어에서 암호를 입력해 문서를 연 뒤 암호 없는 사본으로 저장하고, 그 사본을 업로드해 주세요. 병합이 끝난 결과물에는 원본의 암호가 적용되지 않습니다.

병합하면 화질이나 품질이 떨어지나요?

아니요. 이 도구는 각 문서의 페이지를 원본 그대로 새 문서에 복사하는 방식이라 이미지 재압축이나 텍스트 변환이 일어나지 않습니다. 글자 선명도, 이미지 해상도, 페이지 크기 모두 원본과 동일하게 유지됩니다.

파일 순서를 잘못 올렸는데 다시 올려야 하나요?

다시 올릴 필요 없습니다. 선택된 파일 목록에서 항목을 드래그하면 순서가 바로 바뀝니다. 만약 이미 병합을 마친 뒤에 페이지 단위로 순서를 손보고 싶다면 PDF 페이지 재정렬 도구를 이용하시면 됩니다.

스마트폰에서도 쓸 수 있나요?

네, 스마트폰과 태블릿 브라우저에서도 사용할 수 있습니다. 점선 영역을 탭하면 기기의 파일 앱에서 PDF를 선택할 수 있습니다. 다만 파일이 많거나 용량이 크면 모바일 기기의 메모리 한계로 처리가 오래 걸릴 수 있으니, 대량 작업은 PC에서 하시는 편이 쾌적합니다.

업로드한 파일이 서버로 전송되나요?

전송되지 않습니다. 파일 읽기부터 병합, 다운로드 파일 생성까지 전 과정이 사용자의 브라우저 안에서만 처리됩니다. 인터넷 연결은 페이지와 라이브러리를 처음 불러올 때만 필요하며, 계약서나 개인정보가 담긴 문서도 외부로 나가지 않아 안심하고 사용할 수 있습니다.

원본의 북마크나 링크는 유지되나요?

페이지 안의 내용(텍스트, 이미지, 페이지 내 링크)은 그대로 유지됩니다. 다만 문서 차원의 목차(북마크) 구조는 병합 과정에서 새 문서로 옮겨지지 않을 수 있으니, 목차가 중요한 문서라면 병합 후 결과물을 한 번 열어 확인해 보시기 바랍니다.

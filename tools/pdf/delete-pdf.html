PDF 페이지 삭제란?

PDF 페이지 삭제는 문서에서 불필요한 페이지를 골라서 제거하는 기능입니다. 썸네일 미리보기를 통해 각 페이지의 내용을 확인하고, 클릭 한 번으로 삭제 대상을 지정할 수 있습니다. 별도의 소프트웨어 설치 없이 브라우저에서 바로 사용 가능하며, 파일이 외부 서버로 전송되지 않아 보안 면에서도 안심할 수 있습니다.

무엇을 할 수 있나요?

  • 모든 페이지의 축소판 미리보기 자동 생성
  • 클릭으로 개별 페이지 선택/해제
  • 전체 선택, 선택 해제, 홀수/짝수 페이지 빠른 선택 버튼
  • 삭제 대상 페이지 수 실시간 표시
  • 최소 1페이지는 반드시 남겨야 하는 안전 장치 내장
  • 브라우저 내 100% 처리 -- 서버 업로드 없음

사용 가이드

  1. PDF 파일을 드래그하거나 클릭하여 업로드합니다
  2. 페이지 썸네일이 그리드 형태로 표시됩니다
  3. 삭제하고 싶은 페이지를 클릭하면 빨간색 테두리와 X 표시가 나타납니다
  4. 빠른 선택 버튼(전체 선택, 홀수, 짝수)을 활용하면 한 번에 여러 페이지를 선택할 수 있습니다
  5. '선택한 페이지 삭제 후 다운로드' 버튼을 클릭합니다
  6. 선택한 페이지가 제거된 새 PDF가 자동으로 다운로드됩니다

추천 활용법

  • 불필요한 표지, 목차, 광고 페이지 제거
  • 스캔 문서에서 빈 페이지나 잘못 스캔된 페이지 삭제
  • 긴 보고서에서 필요한 부분만 추출하여 공유
  • 수업 자료에서 특정 챕터만 남기기
  • 계약서나 양식에서 해당되지 않는 페이지 제거 후 제출

참고사항

  • 삭제 전 반드시 원본 파일을 백업해 두세요
  • 모든 페이지를 선택하면 삭제가 불가합니다. 최소 1페이지는 남아 있어야 합니다
  • 홀수/짝수 선택 기능은 양면 스캔 문서에서 한쪽 면만 제거할 때 유용합니다
  • 페이지 수가 많은 문서는 썸네일 생성에 시간이 걸릴 수 있습니다
  • 삭제 후 페이지 번호는 자동으로 재배열됩니다

Q&A

Q: 삭제한 페이지를 복구할 수 있나요?

다운로드된 파일에서는 복구할 수 없습니다. 원본 파일은 변경되지 않으므로 원본을 보관해 두세요.

Q: 비밀번호가 걸린 PDF도 처리할 수 있나요?

암호화된 PDF는 읽을 수 없어 처리가 불가합니다. 먼저 비밀번호를 해제해 주세요.

Q: 삭제 후 남은 페이지의 내용이 바뀌나요?

아닙니다. 남은 페이지의 텍스트, 이미지, 서식 등은 원본 그대로 유지됩니다.

Q: 파일 크기 제한이 있나요?

특별한 크기 제한은 없지만, 매우 큰 파일(100MB 이상)은 브라우저 메모리에 따라 처리가 느려질 수 있습니다.

Q: 페이지 순서를 변경하고 싶으면 어떻게 하나요?

페이지 순서 변경은 PDF 페이지 재정렬 도구를 이용해 주세요. 이 도구는 삭제 전용입니다.

Q: 모바일에서도 사용할 수 있나요?

네, 모바일 브라우저에서도 정상적으로 동작합니다. 화면이 작아도 썸네일 그리드가 자동으로 조절됩니다.

Q: 여러 파일을 한 번에 처리할 수 있나요?

한 번에 하나의 PDF만 처리할 수 있습니다. 작업 완료 후 '파일 변경' 버튼으로 다음 파일을 불러오세요.

Q: 파일이 서버에 업로드되나요?

아닙니다. 모든 처리는 사용자의 브라우저에서 이루어지며, 파일이 외부 서버로 전송되지 않습니다.

Q: 어떤 페이지인지 구분이 어려울 때는?

각 썸네일 아래에 페이지 번호가 표시됩니다. 썸네일의 내용을 확인하여 삭제 대상을 정확히 선택하세요.

Q: 삭제 작업 후 파일 이름은 어떻게 되나요?

원본 파일명 뒤에 '_edited'가 붙어서 다운로드됩니다. 예를 들어 'report.pdf'는 'report_edited.pdf'가 됩니다.

PDF를 이미지로 변환

PDF 문서의 각 페이지를 이미지 파일로 변환합니다. 프레젠테이션, SNS 공유, 썸네일 생성 등에 유용합니다.

사용 방법

  1. PDF 파일을 드래그하거나 클릭하여 업로드합니다
  2. 출력 형식(PNG/JPG)과 해상도를 선택합니다
  3. 변환할 페이지를 선택합니다
  4. '다운로드' 버튼을 클릭하여 이미지를 저장합니다

자주 묻는 질문

Q: PNG와 JPG 중 어떤 형식을 선택해야 하나요?

텍스트가 많거나 투명 배경이 필요하면 PNG, 사진이 많고 파일 용량을 줄이고 싶다면 JPG를 선택하세요.

Q: 해상도(DPI)는 어떻게 설정하나요?

웹용은 72~150 DPI, 인쇄용은 300 DPI 이상을 권장합니다. 높은 DPI일수록 선명하지만 파일 크기가 커집니다.

Q: 여러 페이지를 한 번에 변환할 수 있나요?

네, 모든 페이지 또는 원하는 페이지만 선택하여 일괄 변환할 수 있습니다. ZIP 파일로 다운로드됩니다.

Q: 변환된 이미지의 품질이 떨어지나요?

PDF의 벡터 요소는 선택한 해상도로 래스터화됩니다. 고해상도를 선택하면 원본에 가까운 품질을 얻을 수 있습니다.

모든 처리는 브라우저에서 이루어지며 파일이 서버로 전송되지 않습니다.
